# Poster Images

This directory contains the poster images for the work opportunities section.

## Required Images

Please add the following images to this directory:

1. **azerbaijan-hiring.jpg** - The Azerbaijan hiring poster
2. **norway-work-permit.jpg** - The Norway work permit poster  
3. **europe-work-permit.jpg** - The Spain/Croatia/Italy work permit poster

## Image Requirements

- **Format**: JPEG or PNG
- **Recommended dimensions**: 1080x1080px (square aspect ratio)
- **File size**: Keep under 500KB for optimal loading
- **Quality**: High resolution for clarity when expanded

## How to Add Images

1. Save your poster images with the exact filenames listed above
2. Place them in this directory: `/public/images/posters/`
3. The website will automatically display them in the poster gallery

## Using Custom URLs

If you prefer to use external URLs or different filenames, edit the `posters` array in:
`/src/components/PosterGallery.tsx`

Replace the `imageUrl` values with your preferred paths or URLs.
